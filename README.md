A Map Tracker project made for TLoZ: Phantom Hourglass Archipelago randomizer, based on Zobee's Item Tracker.



This plans to feature :

* small improvements from the item trackers (changing how progressive items work, how dungeons are noted, preventing dupe tracking when reconnecting the autotracker, etc...)
* Map tracker in Poptracker (currently only Universal Tracker supports map tracking)
* Autotracking from AP server



The project will grow slowly, so please be patient.

Here is how I plan to make development for this project : 



Phase 1 : have a solid item tracker and autotracker as a basis.

Phase 2 : implement map tracking. Only overworld map at first. Entrance Randomisation will not be supported yet.

Phase 3 : implement multiple maps with toggleable auto-tabbing, and progressive implementation of ER.
